# WordPress Gutenberg Plugin Boilerplate

We use this as a starter template to build our plugins.

# Requirements

Node 8+

# Setup

Download this folder in your plugins directory

```
cd /wp-content/plugins/wp-gutenberg-plugin-boilerplate/
npm install
```
To start developing
```
npm start
```

To build
```
npm run build
```
